# Yellow Taxi Trip Analysis – Lynx Use Case

This repository contains an exploratory analysis of TLC NYC Yellow Taxi trip records from 2018 through January 2023, developed as part of the Lynx recruitment process for a Data Analytics role. The project examines demand patterns, revenue dynamics, operational behavior, and selected data quality considerations relevant for business reporting and decision-making.

## Repository Structure

- **Datasources/**  
  Folder containing the input and output data sources used throughout the analysis. CSV files are not tracked in the repository, which is why they may not be visible here.

- **Images/**  
  Folder containing the Power BI report used to support chart creation, as well as images later embedded in the Jupyter Notebook deliverable.

- **Yellow-Taxi-Analysis.ipynb**  
  Main Jupyter Notebook containing the full exploratory analysis, findings, and final deliverable.

- **Yellow-Taxi-Analysis.html**  
  HTML export of the notebook. This version is recommended if images do not render correctly in the Jupyter Notebook.

- **Executive Summary - One Pager.docx**  
  Standalone Word document containing the executive summary only, intended for stakeholders who do not require the full analytical walkthrough.

## Scope of the Analysis

The analysis covers TLC NYC Yellow Taxi trip records from 2018 to January 2023 and combines them with the Taxi Zone Lookup file to enrich location-based analysis. The work focuses on identifying trip patterns, segment differences, demand and revenue evolution, and data quality issues that may affect KPI interpretation.

## Notes

Some source files are intentionally excluded from version control, particularly large CSV datasets. The notebook and HTML deliverable are the main artifacts to review, with the HTML version serving as the preferred fallback when notebook rendering issues occur.
